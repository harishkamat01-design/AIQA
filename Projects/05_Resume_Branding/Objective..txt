##Objective — Job Tracker AI
Build a local-first Kanban-style Job Application Tracker as a single-page React application. 
All data persists in the browser using IndexedDB — no backend, no authentication, no API calls.

##North Star: 
A clean, professional Kanban board to track job applications through every stage (Wishlist → Applied → Follow-up → Interview → Offer → Rejected), with drag-and-drop, search, import/export, and light/dark mode.
