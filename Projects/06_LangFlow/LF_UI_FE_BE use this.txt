Here are the steps to connect a UI React frontend Vercel website to a Backend Supabase and database:

Step 1: Create a Supabase account and project

Go to the Supabase website and sign up for an account.
Create a new project by clicking on the "New Project" button.
Fill in the project details, such as project name, password, and database name.
Click on the "Create Project" button to create the project.
Step 2: Create a database in Supabase

In the Supabase dashboard, click on the "SQL Editor" tab.
Create a new database by running the following command:
CREATE DATABASE mydatabase;
Replace "mydatabase" with the name of your database.
Step 3: Create tables in the database

In the SQL Editor, create tables for your data by running the following command:
CREATE TABLE mytable (id SERIAL PRIMARY KEY, name VARCHAR(255), email VARCHAR(255));
Replace "mytable" with the name of your table, and "id", "name", and "email" with the column names and data types.
Step 4: Create a Supabase API key

In the Supabase dashboard, click on the "Settings" tab.
Click on the "API" tab.
Click on the "Create API Key" button.
Fill in the API key details, such as key name and permissions.
Click on the "Create API Key" button to create the API key.
Step 5: Install the Supabase JavaScript library in your React project

Run the following command in your terminal:
npm install @supabase/supabase-js
Import the Supabase library in your React component:
import { createClient } from '@supabase/supabase-js';
Step 6: Initialize the Supabase client in your React component

Create a new instance of the Supabase client:
const supabaseUrl = 'https://your-supabase-url.supabase.co';
const supabaseKey = 'your-supabase-api-key';
const supabase = createClient(supabaseUrl, supabaseKey);
Step 7: Use the Supabase client to interact with your database

Use the Supabase client to perform CRUD (Create, Read, Update, Delete) operations on your database tables.
For example, to fetch data from a table:
const { data, error } = await supabase.from('mytable').select('*');
Step 8: Deploy your React application to Vercel

Create a new Vercel account and project.
Link your GitHub repository to Vercel.
Configure your Vercel project settings, such as environment variables and build settings.
Deploy your React application to Vercel.
Step 9: Configure environment variables in Vercel

In your Vercel project settings, click on the "Environment Variables" tab.
Add your Supabase API key and URL as environment variables.
Use these environment variables in your React component to initialize the Supabase client.
Here is an example of how you can use the Supabase client in your React component:

jsx
import { createClient } from '@supabase/supabase-js';

const supabaseUrl = process.env.SUPABASE_URL;
const supabaseKey = process.env.SUPABASE_KEY;

const supabase = createClient(supabaseUrl, supabaseKey);

const MyComponent = () => {
  const [data, setData] = useState([]);

  useEffect(() => {
    const fetchData = async () => {
      const { data, error } = await supabase.from('mytable').select('*');
      if (error) {
        console.error(error);
      } else {
        setData(data);
      }
    };
    fetchData();
  }, []);

  return (
    <div>
      {data.map((item) => (
        <div key={item.id}>{item.name}</div>
      ))}
    </div>
  );
};
Note: Make sure to replace the
SUPABASE_URL
and
SUPABASE_KEY
environment variables with your actual Supabase URL and API key.
