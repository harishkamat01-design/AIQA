About this demo
This portal's profile card is wired to GoREST, a free public REST API used for prototyping. On load, this page performs a live GET request to https://gorest.co.in/public/v2/users and displays one of the returned demo records above as a stand-in customer profile — showing how the portal could integrate with a real account/CRM API. Fields shown (name, email, gender, status) come directly from that live response, not from a local mock.
