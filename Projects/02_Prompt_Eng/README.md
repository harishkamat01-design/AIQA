# Prompt Engineering

This repository hosts a collection of prompt templates, test‑case generators, and frameworks for building robust LLM‑powered applications.

## Structure

- **`Project1_TC_Generation/`** – Prompt templates and generated test cases.  
- **`Project2_Framework/`** – Framework skeletons for Selenium, Playwright, and API testing.  
- **`TestPlan_Local_LLM_VS_Code/`** – Sample test plans and PRD‑to‑test‑plan prompts.

## Usage

1. Copy a template from `templates/` into your own project.  
2. Run the corresponding generator script to produce test cases.  
3. Adjust the prompts to match your domain language.
